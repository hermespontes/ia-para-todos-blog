# IA para Todos

Blog e base de conhecimento do projeto **IA para Todos** (popularização de IA para pessoas não-técnicas).

## Objetivo
- Explicar IA de forma simples, prática e sem hype
- Ensinar uso seguro e responsável
- Guiar pessoas comuns a resolver problemas reais com IA

## Estrutura
- Conteúdo do blog em `_posts/`
- Layout e assets em `assets/`
- “Agentes” (prompts + runner) em `agents/`

## Publicação (GitHub Pages)
- Este repositório foi pensado para rodar em **GitHub Pages**.
- Para publicar: criar o repo e habilitar Pages (branch `main`, pasta `/root`).
